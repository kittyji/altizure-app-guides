# 支持的无人机型号

#### iOS 3.7.2 版目前支持的机型

* Mavic Pro
* Inspire 1 \(X3 / X5 / X5R\)
* Phantom 3 \(Pro / Advanced / 4K / Standard\)
* Phantom 4 \(4 / 4 Adv / 4 Pro\)
* M100, M600
* A2 / A3 + LB2
* \(测试中\) Inspire 2
* \(测试中\) N3 + LB2

* Mavic Pro
* Inspire 1 \(X3 / X5 / X5R\)
* Phantom 3 \(Pro / Advanced / 4K / Standard\)
* Phantom 4 \(4 / 4 Adv / 4 Pro\)
* M100, M600
* A2 / A3 + LB2
* \(测试中\) Inspire 2
* \(测试中\) N3 + LB2
